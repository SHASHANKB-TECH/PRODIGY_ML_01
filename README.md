🏠 House Price Prediction using Linear Regression

A Machine Learning project that predicts house prices using Linear Regression based on features such as living area, number of bedrooms, full bathrooms, and half bathrooms.

🔧 Technologies Used:

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

📊 Features:

* Data preprocessing
* Feature scaling
* Train-test split
* Linear Regression model training
* Model evaluation using R², MAE, and RMSE
* Visual analytics dashboard

📈 Model Performance:

* R² Score: 0.63
* MAE: $36,019
* RMSE: $53,018

This project demonstrates the end-to-end workflow of a regression-based Machine Learning model including data analysis, training, evaluation, and visualization.
